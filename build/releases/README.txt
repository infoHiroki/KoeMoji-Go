🎵 KoeMoji-Go v1.5.1 - 音声ファイル自動文字起こしツール

音声・動画ファイルを自動で日本語文字起こしするアプリケーションです！
シングルバイナリなので、ダウンロードしてすぐに使えます。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📥 ダウンロード - お使いのOSに合わせて選んでください

🪟 Windows版
　　ファイル名: KoeMoji-Go_Win.zip
　　対象: Windows 10/11 (64bit)

🍎 Mac版 (Apple Silicon)
　　ファイル名: KoeMoji-Go_Mac.tar.gz
　　対象: M1/M2 CPU搭載のMac

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📋 使用前に必要なもの

⚠️ 重要: Python 3.8以上が必要です

🔍 Pythonがインストールされているか確認:
　　コマンドプロンプト(Windows)またはターミナル(Mac)で
　　python --version または python3 --version

❌ Pythonがない場合:
　　https://www.python.org/downloads/ からダウンロード
　　Windows: インストール時に「Add Python to PATH」をチェック！

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚡ 使い方（3ステップで完了！）

1️⃣ ダウンロード・展開
　　上記のファイルをダウンロードして展開

2️⃣ 実行
　　🪟 Windows: koemoji-go.exe をダブルクリック
　　🍎 Mac: ターミナルで ./koemoji-go

3️⃣ 音声ファイルを置くだけ
　　📁 input フォルダに音声ファイル(MP3, WAV等)を置く
　　⏳ 自動的に処理が開始されます
　　📄 結果は output フォルダに保存されます

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🎛️ その他の機能

🎤 録音機能: r キーで録音開始/停止
🤖 AI要約: OpenAI APIキーを設定すると自動要約も可能
🖥️ GUI/TUI: グラフィカル画面とターミナル画面を選択可能

対応ファイル:
📼 音声: MP3, WAV, M4A, FLAC, OGG, AAC
🎬 動画: MP4, MOV, AVI

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔧 困った時は

❓ 起動しない
　　→ Pythonがインストールされているか確認

❓ 処理されない
　　→ 対応ファイル形式か確認
　　→ input フォルダに正しくファイルが置かれているか確認

❓ 処理が遅い
　　→ 設定で小さなモデルに変更可能

❓ その他のトラブル
　　→ 詳細なトラブル解決ガイドをご覧ください
