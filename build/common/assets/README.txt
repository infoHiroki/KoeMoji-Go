================================================================================
KoeMoji-Go v1.7.0
音声・動画ファイル自動文字起こしツール（macOS APP版）
================================================================================

■ インストール方法

1. このDMGウィンドウで、KoeMoji-Go.appをApplicationsフォルダにドラッグ&ドロップ
2. コピー完了後、DMGウィンドウを閉じて「KoeMoji-Go」ボリュームを取り出す

【重要】DMG上の.appを直接ダブルクリックすると「壊れている」エラーが出ます。
       必ずApplicationsフォルダにコピーしてから起動してください。

--------------------------------------------------------------------------------

■ 初回起動（重要）

通常のダブルクリックでは起動できません。以下の手順で起動してください：

1. Applicationsフォルダを開く
2. KoeMoji-Go.appを「右クリック」（または Control + クリック）
3. メニューから「開く」を選択
4. 警告ダイアログが表示される：
   "KoeMoji-Go.app"は、開発元が未確認のため開けません。
5. ダイアログの「開く」ボタンをクリック
6. マイクアクセス許可ダイアログが表示されたら「OK」をクリック
7. 2回目以降は通常通りダブルクリックで起動可能

【なぜこの警告が出るのか？】
KoeMoji-Goはコード署名を行っていないためです。コード署名にはApple 
Developer Program（年間13,800円）の加入が必要で、現在はコストの関係で
実施していません。

【安全性について】
・ソースコードはGitHubで公開されています
  https://github.com/infoHiroki/KoeMoji-Go
・公式のGitHub Releasesからのダウンロードを推奨します
・マルウェアやウイルスは含まれていません

--------------------------------------------------------------------------------

■ 基本的な使い方

1. KoeMoji-Goを起動
2. ~/Documents/KoeMoji-Go/input/ フォルダに音声・動画ファイルを配置
3. 自動的に文字起こしが開始される
4. ~/Documents/KoeMoji-Go/output/ フォルダに結果が保存される
5. 処理済みファイルは ~/Documents/KoeMoji-Go/archive/ に移動

【対応ファイル形式】
・音声: MP3, WAV, M4A, FLAC, OGG, AAC
・動画: MP4, MOV, AVI

--------------------------------------------------------------------------------

■ データ保存場所

全てのデータは以下のディレクトリに保存されます：

~/Documents/KoeMoji-Go/
├── input/         処理対象のファイルを配置
├── output/        文字起こし結果が保存される
├── archive/       処理済みファイルが移動される
├── config.json    設定ファイル
└── koemoji.log    ログファイル

--------------------------------------------------------------------------------

■ 録音機能

GUI画面の「録音開始」ボタンで録音できます。
録音停止後、自動的にinputフォルダに保存され、文字起こしが開始されます。

初回録音時にマイクアクセス許可を求められますので「OK」をクリックしてください。

--------------------------------------------------------------------------------

■ アンインストール方法

1. /Applications/KoeMoji-Go.appを削除
2. ~/Documents/KoeMoji-Go/フォルダを削除

全てのデータが完全に削除されます。

--------------------------------------------------------------------------------

■ トラブルシューティング

【起動できない】
1. 「初回起動（重要）」の手順を実施したか確認
2. 右クリック→「開く」で起動を試す
3. ターミナルから実行してエラーメッセージを確認：
   /Applications/KoeMoji-Go.app/Contents/MacOS/koemoji-go

【Python/FasterWhisperエラー】
ターミナルで以下を実行：
  python3 --version        # Python 3のインストール確認
  pip3 install faster-whisper  # FasterWhisperのインストール

【処理が始まらない】
1. ファイル形式が対応しているか確認
2. デバッグモードで起動してログを確認：
   /Applications/KoeMoji-Go.app/Contents/MacOS/koemoji-go --debug
