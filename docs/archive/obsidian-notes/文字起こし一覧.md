## 全ノート一覧

```dataview
TABLE date AS "日付", source AS "元ファイル", whisper_model AS "モデル", has_summary AS "要約"
FROM "コエモジ"
WHERE type = "transcription"
SORT date DESC
```

## 要約ありのみ

```dataview
TABLE date AS "日付", source AS "元ファイル", summary_model AS "要約モデル"
FROM "コエモジ"
WHERE has_summary = true
SORT date DESC
```

## タグ別

```dataview
LIST
FROM "コエモジ"
WHERE type = "transcription"
GROUP BY tags
```
